### Criando o projeto:
    - yarn init -y

    - yarn add express body-parser sqlite3 ejs

### Adicionando o Nodemon ao projeto
    - yarn add nodemon --dev 
    ou
    - npm install nodemon --save-dev

#### Configurando Nodemon
    'package.json'
    
    "scripts": {
        "start": "node app.js",
        "dev": "nodemon app.js"
    }

#### Rodando o projeto
    - yarn dev
    ou
    - npm run dev
